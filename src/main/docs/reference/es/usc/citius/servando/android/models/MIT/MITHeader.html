public void Parse (String HeaderString)

Analiza la cadena de texto pasada como argumento, considerándola como el texto completo separado en líneas de un archivo de cabecera ".hea". En base al contenido, se establecen las diferentes propiedades de esta instancia. La cadena puede contener líneas de comentarios, que serán convenientemente ignoradas.
